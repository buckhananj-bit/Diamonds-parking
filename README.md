This repository hosts the customer-facing pay-to-park pages for **Diamonds Parking**.

Each parking lot (zone) has its **own page** with:
- Zone + address displayed at the top
- License plate entry
- Hourly rate dropdown (or event-day flat rate)
- Mobile-friendly payment UI

The site is deployed as a **static site on Vercel** and is accessed via **QR codes and SMS links** posted at each lot.

---

## Tech Stack

- Static HTML / CSS / JavaScript
- No framework
- Hosted on Vercel
- No build step required
